# esphome-projects

Collection of ESPHome projects, each with its own directory containing the YAML configuration and project documentation.

## Projects

### [Pet Feeder](./pet-feeder/) — WENHOME 3L Automatic Cat Feeder (YB_5L)

Converted a WENHOME 3L pet feeder to local Home Assistant control by replacing the Tuya WBR3 Wi-Fi module with an ESP32-S3. Runs feeding schedules, manual feeds, and telemetry entirely on the local LAN — no cloud dependency.
